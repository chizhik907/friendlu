# ❤️ Сердечные знакомства

Полноценный сайт знакомств с REST API на Python/Flask и адаптивным фронтендом на HTML/CSS/JS.

## 📋 Функционал

- ✅ Регистрация и авторизация пользователей
- ✅ Просмотр анкет противоположного пола
- ✅ Лайки и дизлайки с анимацией
- ✅ Взаимные симпатии (Match)
- ✅ Чат между пользователями в реальном времени
- ✅ Редактирование профиля
- ✅ Тестовые пользователи для быстрого старта
- ✅ Адаптивный дизайн для мобильных устройств

## 🛠 Технологии

### Backend
- Python 3.10+
- Flask 2.3.3
- Flask-CORS 4.0.0
- SQLite 3 (встроенная БД)

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript (ES6)

## 📁 Запуск
pip install flask flask-cors
python app.py
